# Reto 001

## Refactoring

Dado el código a continuación, que además funciona, identifique las malas prácticas que contiene y proponga y aplique una refactorización al mismo. Dicha refactorización debe ser hecha mediante *baby steps*, es decir, una corrección por commit.

### Códigos

- [Calculadora de descuentos](CalculadoraDescuentos.java)
- [Conversor de unidades](ConversorUnidades.java)
- [Juego de adivinanza](JuegoAdivinanza.java)
- [Gestor de tareas](GestorTareas.java)

### Entregables

El trabajo debe construirse **a partir de** la rama `entregas` de este repo, y **enviarse a** la rama reto-001 en este mismo repo.

- [ ] **Código refactorizado** de al menos uno de los ejercicios
- [ ] **Historial de commits** visible (una mejora por commit). Este historial de commit, bien hecho, se convierte en:
  - [ ] Documento con las **malas prácticas identificadas** en cada código
  - [ ] Breve **memoria** justificando los cambios realizados
